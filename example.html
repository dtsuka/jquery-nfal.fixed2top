NFAL Fixed to top - jQuery Plugin

よくあるサイドメニューがスクロールに付いてくる奴です。

対象のエレメントに、スクロール中は".f2t_scrolling"、下端までスクロールした時は".f2t_bottomEnd"のクラスを追加します。

$('#menu').fixed2top();

フッターにかぶらないようにする場合

$('#menu').fixed2top({
	paddingBottom:$('#footer').outerHeight(true)
});

options

Name デフォルト値 説明
debugmode false trueでデバッグモード
offsetTop 'auto' 対象エレメントを上から何ピクセルずらして表示するかの設定。
基本的にautoのままで使用
paddingBottom 0 フッターに被せないようにする場合は、ここにフッターの高さを数値指定
fixedClass 'f2t_scrolling' スクロール中に追加されるクラス名
absoluteClass 'f2t_bottomEnd' 下端までスクロールされた時に追加されるクラス名
